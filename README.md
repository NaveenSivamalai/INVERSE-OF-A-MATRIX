Get the input matrix from the user
### Step 3:

### Step 4: 

End the program


## Program:
```
#Program to find the rank of a matrix.

import numpy as np
A=np.array([[2,1,1],[1,1,1],[1,-1,2]])
inverse=np.linalg.inv(A)
print(inverse)
```
